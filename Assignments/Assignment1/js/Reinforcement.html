Reinforcement learning is an area of machine learning inspired by behaviorist psychology, concerned with how software agents ought to take actions in an environment so as to maximize some notion of cumulative reward. In economics and game theory, reinforcement learning may be habituated to expound how equilibrium may arise under bounded rationality.

In machine learning, the environment is typically formulated as a Markov decision process (MDP) as many reinforcement learning algorithms for this solution utilizes dynamic programming techniques. The main distinction between the classical techniques and reinforcement learning algorithms is that the latter do not require cognizance about the MDP and they target immensely colossal MDPs where exact methods become infeasible.

Reinforcement learning differs from standard supervised learning in that correct input/output dyads are never presented, nor sub-optimal actions explicitly redressed. Further, there is a fixate on on-line performance, which involves finding a balance between exploration (of uncharted territory) and exploitation (of current erudition).
